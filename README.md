# 🛒 Zepto_SQL_Inventory_Analysis

## 📌 Project Overview

This project focuses on analyzing Zepto's product inventory dataset using SQL. The objective is to clean the dataset, explore product information, and answer business-oriented questions related to pricing, discounts, stock availability, and product categories.

The project demonstrates how SQL can be used for exploratory data analysis (EDA) and business reporting on retail inventory data.

---

## 🎯 Objectives

- Clean and prepare inventory data.
- Analyze pricing and discount trends.
- Evaluate stock availability.
- Identify high-value products.
- Generate business insights from inventory data.

---

## 🗂️ Dataset

The dataset contains product-level information from Zepto's inventory, including:

- Product Name
- Category
- Selling Price
- Market Price
- Discount Percentage
- Available Quantity
- Stock Status

## 📊 Business Questions Solved

- How many unique product categories exist?
- Which products have the highest discounts?
- Which categories contain the most products?
- Which products are currently out of stock?
- What is the average discount across products?
- Which products have the highest selling price?
- Which categories generate the highest inventory value?
- How many products are available in each category?

---

## 📈 Key Insights

- Identified categories with the largest product assortment.
- Compared selling prices with market prices.
- Analyzed discount distribution across products.
- Identified out-of-stock products.
- Evaluated inventory availability by category.
